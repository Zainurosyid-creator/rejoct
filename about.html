Project html

html adalah bahasa pemograman untuk membuat website

Apa itu HTML?

HTML singkatan dari HyperText Markup Language. Fungsinya buat bikin struktur halaman web kayak judul, paragraf, gambar, link.

Kenapa belajar HTML?

Karena HTML itu dasar dari semua website. Mau bikin web pake React, Vue, WordPress, semuanya tetep pake HTML di belakangnya.

Contoh Tag HTML

  • <h1> sampai <h6> : Untuk judul
  • <p> : Untuk paragraf
  • <a> : Untuk link
  • <img> : Untuk gambar
  • <div> : Untuk membagi bagian

Cara Mulai

1. Buka VS Code atau Notepad
2. Buat file index.html
3. Tulis kode HTML
4. Buka file nya di browser

Tips

Belajar HTML paling gampang kalau langsung praktek. Coba ubah-ubah kode nya dan lihat hasilnya di browser.
